# FAQ - Часто задаваемые вопросы о миграции на Tauri

## Общие вопросы

### Q: Нужно ли переписывать весь фронтенд?
**A:** Нет! Весь React код остается без изменений. Меняется только бэкенд (с Node.js на Rust) и API слой.

### Q: Будут ли работать мои данные (закладки, история)?
**A:** Да, все данные автоматически мигрируют. Формат хранения остался JSON.

### Q: Можно ли вернуться на Electron?
**A:** Да, старые файлы Electron сохранены. Просто переключитесь на старую ветку git.

## Технические вопросы

### Q: Почему Tauri использует системный WebView вместо Chromium?
**A:** Это главное преимущество Tauri - не нужно упаковывать весь Chromium в приложение. Системный WebView уже есть в ОС, что уменьшает размер приложения в 30 раз.

### Q: Какой WebView используется на разных ОС?
**A:**
- **Linux**: WebKit (как в Safari)
- **macOS**: WebKit (нативный)
- **Windows**: Edge WebView2 (Chromium-based)

### Q: Будут ли работать Chrome расширения?
**A:** Нет, Chrome расширения работают только в Chromium. Но можно реализовать собственную систему расширений.

### Q: Как работает AdBlock без Ghostery?
**A:** Используется простой блокировщик на уровне доменов. Можно добавить полноценный AdBlock с EasyList через Rust библиотеки.

### Q: Нужно ли знать Rust для разработки?
**A:** Для изменения фронтенда - нет. Для изменения бэкенда (IPC команды, OAuth, хранилище) - желательно, но код хорошо документирован.

## Производительность

### Q: Насколько быстрее Tauri?
**A:**
- Запуск: в 2-3 раза быстрее
- Потребление памяти: в 3-5 раз меньше
- Размер приложения: в 30 раз меньше

### Q: Почему Tauri быстрее?
**A:**
1. Rust быстрее Node.js
2. Нет overhead от встроенного Chromium
3. Меньше слоев абстракции
4. Лучшая оптимизация компилятора

## Разработка

### Q: Как запустить в режиме разработки?
**A:**
```bash
npm run dev
```

### Q: Как собрать production версию?
**A:**
```bash
npm run build
```

### Q: Где находятся собранные файлы?
**A:** В `src-tauri/target/release/bundle/`

### Q: Как отладить Rust код?
**A:**
```bash
RUST_LOG=debug npm run dev
```

### Q: Как открыть DevTools?
**A:** Нажмите `Ctrl+Shift+I` или `F12` в приложении

## Проблемы и решения

### Q: Ошибка "webkit2gtk not found"
**A:** Установите системные зависимости:
```bash
sudo apt install libwebkit2gtk-4.0-dev build-essential
```

### Q: Ошибка "cargo not found"
**A:** Установите Rust:
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
```

### Q: Приложение не запускается
**A:** Проверьте:
1. Установлен ли Rust: `rustc --version`
2. Установлены ли системные зависимости
3. Выполнен ли `npm install`

### Q: Ошибка компиляции Rust
**A:** Обновите Rust:
```bash
rustup update
```

### Q: Горячие клавиши не работают
**A:** Проверьте, что в `main.rs` вызывается `setup_shortcuts()`

## Функциональность

### Q: Работает ли Google OAuth?
**A:** Да, OAuth реализован через Rust с использованием локального HTTP сервера.

### Q: Работает ли импорт из Chrome/Edge?
**A:** Да, импорт закладок работает так же как в Electron версии.

### Q: Работают ли загрузки?
**A:** Да, загрузки управляются через системный WebView.

### Q: Работает ли восстановление сессий?
**A:** Да, сессии сохраняются и восстанавливаются автоматически.

### Q: Работают ли workspaces?
**A:** Да, вся логика workspaces осталась на фронтенде без изменений.

## Безопасность

### Q: Безопаснее ли Tauri чем Electron?
**A:** Да, по нескольким причинам:
1. Rust обеспечивает memory safety
2. Меньше поверхность атаки (нет Node.js runtime)
3. Allowlist API по умолчанию
4. Меньше зависимостей

### Q: Как работает IPC в Tauri?
**A:** Через Tauri Commands - безопасный механизм вызова Rust функций из JavaScript.

### Q: Можно ли использовать Node.js модули?
**A:** Нет, в Tauri нет Node.js runtime. Но большинство функций можно реализовать на Rust.

**A:** Tauri имеет встроенный updater, настраивается в `tauri.conf.json`

- Microsoft Store (Windows)

- Snap Store (Linux)
- Flathub (Linux)

## Миграция данных

### Q: Где хранятся данные?
**A:**
- **Linux**: `~/.local/share/xolo-browser/`
- **macOS**: `~/Library/Application Support/xolo-browser/`
- **Windows**: `%APPDATA%\xolo-browser\`

### Q: Совместимы ли данные между Electron и Tauri версиями?
**A:** Да, формат данных одинаковый (JSON файлы).

### Q: Как перенести данные вручную?
**A:** Скопируйте файлы из старой папки данных в новую.
